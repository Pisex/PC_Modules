Примеры использования
```
"outcomes"
{
    //Пример выдачи premium статуса (группа|время (0 - Навсегда)|время продления, если группа такая у игрока используется)
    //Выдастся premium `star` на 1час(3600 секунд) если у игрока её нету, а если есть, то продление на 30минут(1800 секунд)
    "premium_star"       "premium_group;star|3600|1800"
}
```
