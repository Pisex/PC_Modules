Примеры использования
```

"outcomes"
{
    //Ферма, пример настройки выдачи фермы, в данном случае нам дадут 2 фермы 1 уровня
    //Число 1 это уровень фермы, число 2 количество ферм этого уровня для выдачи
    "mf_2farm"       "mf_farm;1|2"
    //Биткоины, пример настройки выдачи биткоинов, в данном случае нам дадут 6 биткоинов
    //10000000 = 1 биткоин
    "mf_6bit"          "mf_bitcoin;60000000"
}
```
